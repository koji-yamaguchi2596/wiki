# サービス概要（Overview）

このWikiは社内のサービス概要やプロセス、開発ルールについての情報をまとめています。

## 目的

- 社内の開発プロセスや運用ルールの共有
- 新メンバーのオンボーディング支援
- ナレッジベースとしての情報集約

## 構成

このWikiは以下の章で構成されています：

### 1. サービス概要（Overview）
サービスの全体概要とビジョンについて説明します。

### 2. プロセス（Process）
開発やデザインに関するプロセスについて説明します。

### 3. 開発（Dev）
技術的な開発ルールや運用方針について説明します。

## 更新方法

このWikiの更新は以下の手順で行ってください：

1. feature ブランチを作成
2. 変更内容をコミット
3. Pull Request を作成
4. レビュー後にマージ
